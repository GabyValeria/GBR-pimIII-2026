## 📖 Sobre este repositório

Este repositório foi criado com a finalidade de **armazenar, organizar e apresentar os artefatos individuais** desenvolvidos na disciplina **Engenharia de Software Ágil Aplicada**, relacionados ao projeto do **PIM III**.

Seu objetivo é servir como um espaço complementar ao repositório principal do projeto, reunindo conteúdos acadêmicos voltados à aplicação prática de **métodos ágeis**, especialmente no contexto de planejamento, organização e acompanhamento do desenvolvimento do sistema.

## 🧩 Relação com o PIM III
